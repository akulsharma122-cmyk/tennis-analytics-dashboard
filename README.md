# Tennis Analytics Dashboard

An interactive Tennis Analytics Dashboard built using Python, SQLite, SQL and Streamlit.

## Features

- Homepage Dashboard
- Competitor Search
- Competitor Details
- Country Analysis
- Venue Analysis
- Competition Analysis

## Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- SQL

## Dataset

This project uses SportRadar Tennis API data including:

- Competitors
- Competitor Rankings
- Competitions
- Categories
- Complexes
- Venues
